// Capacidades

Todo lo que JARVIS hace por ti

Diseñado para quienes quieren un asistente real — no otro chatbot. JARVIS escucha, piensa y actúa directamente en tu sistema.

🎙️

Voz en Tiempo Real

Habla directamente con JARVIS a través de Gemini Live WebSocket. Respuesta instantánea, transcripción simultánea y síntesis de voz nativa.

⚡

Control del Sistema

Abre aplicaciones, ejecuta comandos PowerShell, busca archivos y gestiona procesos con solo hablar. JARVIS entiende tu sistema.

🌐

Búsqueda Web

Obtén información actualizada de internet sin salir de la app. JARVIS busca, lee y te resume lo relevante en segundos.

🔒

100% Privado

Sin servidor externo, sin login, sin suscripción. Tu API key de Gemini va directo desde tu máquina a Google. Cero intermediarios.

🧠

Memoria Contextual

JARVIS recuerda tu nombre, tu SO, y el historial de la conversación activa. Instrucciones de sistema personalizables desde la app.

🖥️

Interfaz Holográfica

HUD tipo Iron Man con Arc Reactor animado, burbujas de conversación fluidas y efecto typewriter. Una experiencia visual única.

// Proceso

Listo en 3 pasos

01

Descarga e instala

Un instalador NSIS simple de un solo clic. Sin dependencias extras. Solo Windows 10/11.

02

Ingresa tu API Key

Crea una API Key gratis en Google AI Studio y pégala en la pantalla de configuración de JARVIS.

03

Habla con JARVIS

Presiona el botón de micrófono o usa atajos de teclado. Tu asistente de IA personal está listo.

// Requisitos

¿Qué necesito?

Windows 10 / 11 64-bit, cualquier edición
API Key de Gemini Gratis en aistudio.google.com
Micrófono Para control por voz (opcional)
Conexión a internet Solo para llamadas a Gemini API

Estabilidad, limpieza y refactor de arquitectura
